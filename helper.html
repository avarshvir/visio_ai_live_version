Getting Started

Welcome to Visio AI! This software is designed to make data analysis and machine learning simple and interactive.

  • Sign up or log in to access your dashboard.
  • Use the left-hand menu to navigate through the tools and options.
  • Start by uploading your dataset in supported formats like CSV, Excel, or text files.

Uploading Data

Follow these steps to upload and manage your dataset:

  • Click on the "Upload Data" button in the sidebar.
  • Select the file type (CSV, Excel, or text) and browse for the file.
  • Preview the dataset to ensure it is loaded correctly.
  • Handle missing values automatically or manually using the options provided.

Data Visualization

Visualize your dataset with dynamic charts and graphs:

  • Click on "Generate Plots" in the menu.
  • Select the target variable and the type of plot (e.g., bar, scatter, line).
  • Customize the independent variables for comparison.
  • Download the visualization as an image.

Machine Learning

Train models and make predictions:

  • Go to "Train Your Model" and select the target variable.
  • Split the dataset into training and testing sets with adjustable parameters.
  • Choose a machine learning algorithm and train the model.
  • View predictions and analysis in a pop-up window.

Additional Tools

Enhance your experience with these features:

  • Use the calculator for quick calculations while working on your data.
  • Leverage the notepad to jot down important notes or observations.
  • Access the Jupyter Notebook viewer to review and run Python scripts.
  • Generate comprehensive reports with the built-in report generator.
  • Utilize the AI chatbot for assistance and queries about the software.
