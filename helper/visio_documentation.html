Overview

Visio AI is an advanced data analysis and machine learning platform developed by Jaiho Digital, designed to simplify complex data science tasks for researchers, analysts, and machine learning enthusiasts.

Getting Started

System Requirements

  • Operating System: Windows, macOS, Linux
  • Python 3.8+
  • Streamlit
  • Required Python Libraries:
    • pandas
    • numpy
    • scikit-learn
    • seaborn
    • matplotlib
    • plotly

Installation

  1. Clone the Visio AI repository: 
    git clone https://github.com/jaihodigital/visio-ai.git
cd visio-ai
  2. Create a virtual environment: 
    python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
  3. Install dependencies: 
    pip install -r requirements.txt
  4. Run the Streamlit app: 
    streamlit run app.py

Key Features

1. Data Upload and Missing Value Handling

  • Upload datasets in CSV, Excel, etc.
  • Automatic and Manual Missing Value Handling

2. Visualization Tools

  • Generate Pairplots
  • Create various plot types (Scatter, Bar, Histogram, etc.)

3. Data Operations

  • Split datasets into training/testing
  • Stratified sampling and customizable ratios

4. Machine Learning Algorithms

  • Automated algorithm recommendations
  • Support for Classification, Regression, and Clustering

5. Additional Tools

  • Word Cloud Generator
  • Note-Lite: Built-in note-taking
  • Calculator App
  • Viz AI: Image dataset analysis
  • Viz Report: Comprehensive report generation
